void ACTrie::findNv ( )
inlineoverridevirtual

Set the failure_links and output_links.

Conduct a breadth first traversal over the nodes in the Trie. The final node in the edge list will hold the pointer to the node which the search() algorithm moves to when no match is found. nullptr signifies that search() should resume at root. Otherwise, update the final node.next to point to the appropriate node in the trie.
