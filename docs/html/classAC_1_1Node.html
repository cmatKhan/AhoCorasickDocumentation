◆ edgeSearch()

Edge * AC::Node::edgeSearch ( char  c)
inline

Search the edge list for a given character.

If an edge with a given char label exists, return the pointer to that node in the Trie. NOTE: a reference will always be returned. Make sure the final edge in the Edgelist stores the trie_node reference to the 'failure link'.

Parameters
ca character with which to search the labels of the edge list
Returns
a reference to a Node. NOTE: a reference to a Node will always be returned. Make sure the last node in the edge list points to the 'failure link'.
