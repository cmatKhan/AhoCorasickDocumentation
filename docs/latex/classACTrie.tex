Set the failure\+\_\+links and output\+\_\+links. 

Conduct a breadth first traversal over the nodes in the Trie. The final node in the edge list will hold the pointer to the node which the \mbox{\hyperlink{classACTrie_af99bf3d3e8f77150b1a4ca169f819fc4}{search()}} algorithm moves to when no match is found. nullptr signifies that \mbox{\hyperlink{classACTrie_af99bf3d3e8f77150b1a4ca169f819fc4}{search()}} should resume at root. Otherwise, update the final node.\+next to point to the appropriate node in the trie.
