Set the failure\+\_\+links and output\+\_\+links. 

Conduct a breadth first traversal over the nodes in the Trie. The final node in the edge list will hold the pointer to the node which the \mbox{\hyperlink{classAC_1_1AhoCorasickTrie_a7dc543e54a96ebc66d7a560c8970c6ee}{search()}} algorithm moves to when no match is found. nullptr signifies that \mbox{\hyperlink{classAC_1_1AhoCorasickTrie_a7dc543e54a96ebc66d7a560c8970c6ee}{search()}} should resume at root. Otherwise, update the final node.\+next to point to the appropriate node in the trie.
